# 流媒体中的DRM

## 什么是DRM?

Digital rights management \(DRM\) 是一种对数字媒体版权保护的系统化方法，目的就是防止数字媒体的未授权分发和限制消费者播放内容的方式。 对于流媒体而言，DRM就是加密媒体内容，使得未授权用户在没有取得合法媒体密钥的情况下无法播放，或者限制授权用户在限定设备或时间范围内进行播放。
